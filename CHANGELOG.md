## [0.1.0] — 2026-05-21

### Added
- Initial release of the Python SDK
- `ShopifyAnalyzerClient` with synchronous `analyze()` and `analyze_one()` methods
- Support for all 17 input flags of the underlying actor
- 6 example scripts: quickstart, bulk analyze, dropshipper filter, CSV export,
  competitor comparison, change monitoring
